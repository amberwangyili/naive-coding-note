# 深度优先搜索

* 框架

```c++
void dfs(int deep) {
    if (到达边界) {
      // 做一些处理后返回
    } else {
        for(所有可能的选择) {
            dfs(deep + 1);
        }
    }
}
```

* 迷宫模板

  ```c++
  // 对坐标为(x, y)的点进行搜索
  void dfs(int x, int y) {
      if (x,y) 是终点 {
          方案数增加
          return;
      }
      标记(x, y)已经访问
          // 尝试
      for (x, y) 能到达的格子(tx, ty) {
          if (tx, ty) 没有访问 {
              dfs(tx, ty);
          }
      }
      
      取消(x, y)访问标记
          //　回溯
  }
  ```

  * 回溯一般取决于，是否必须重复经过某一个点
  * 定义一个全局变量可记录最值, 参数传递当前值
